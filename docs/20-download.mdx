# Download and Install

## Method 1. VRChat Creator Companion (Preferred)

* [Click here to add VRCFury to the VRChat Creator Companion](vcc://vpm/addRepo?url=https%3A%2F%2Fvcc.vrcfury.com)
  * Didn't work? In the VCC, go to Settings > Packages > Add Repository, type `https://vcc.vrcfury.com` and click Add
* In the list, make sure `VRCFury Repo` is checked
* Find your project in the VCC project list, click `Manage Project`, then add VRCFury.
* To update VRCFury later on, you can do so by returning to the `Manage Project` VCC page.

## Method 2. VRCFury Installer (Not Recommended)

* Download and import the [VRCFury Installer](https://vrcfury.com/installer)
* To update VRCFury later on, you can do so in unity at `Tools > VRCFury > Update VRCFury`.

## Method 3. OpenUPM (Not Recommended)

VRCFury is available on the OpenUPM at https://openupm.com/packages/com.vrcfury.vrcfury/

## Troubleshooting

Having trouble installing? Check out [the troubleshooting page](/troubleshooting#script-errors)
